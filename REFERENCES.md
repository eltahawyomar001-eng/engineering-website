## Ключевые паттерны для вдохновения

### Цветовые схемы
- Тёмные фоны (#0a0a0a, #111111, #0f0f0f)
- Оранжевые/красные акценты (#e85d3b, #ff4500, #e63946)
- Белый текст на тёмном фоне
- Минимальное использование цвета, максимум контраста

### Типографика
- Display шрифты для заголовков (Space Grotesk, Neue Montreal, Clash Display)
- Sans-serif для body text (Inter, Graphik, Söhne)
- Крупные размеры заголовков (clamp от 48px до 120px)
- Плотный line-height для драматического эффекта

### Анимации
- Smooth scroll с Locomotive Scroll или Lenis
- GSAP для сложных timeline анимаций
- Parallax эффекты на скролле
- Hover state transitions (300-500ms)
- Page transitions с fade/slide эффектами

### Компоненты
- Custom cursor (увеличение при наведении на интерактивные элементы)
- Grid-based портфолио с hover reveal
- Sticky headers с изменением при скролле
- Modal popups для детального просмотра проектов
- Progress indicators и scroll progress bars

### Навигация
- Burger меню на мобильных с fullscreen overlay
- Numbered navigation items
- Hover underline эффекты
- Active state indicators

### Анимация
- GSAP (GreenSock Animation Platform)

### CSS
- CSS Custom Properties
- CSS Grid + Flexbox

- CSS Scroll-snap

### JavaScript
- Intersection Observer API
- Scroll-triggered animations
- Image lazy loading
- Smooth scrolling

## Заметки по дизайну

**Для инжиниринговой компании важно:**
1. Демонстрация экспертизы через кейсы
2. Профессиональный, но не скучный визуал
3. Чёткая структура услуг и процессов
4. Доверие через цифры и факты
5. Простой путь к контакту

**Что избегать:**
- Шаблонные решения
- Переизбыток анимаций
- Сложная навигация
- Медленная загрузка
- Мелкий нечитаемый текст
